TO GET YOUR IPA (3 STEPS)
═══════════════════════════════════════════════════════════════════════════

STEP 1: Create GitHub repo or use existing
  • New: Create "synth" repo on github.com (your account)
  • Or: Use bward-dev1 org if you have write access

STEP 2: Push this code
  $ git remote add origin https://github.com/YOUR_USERNAME/synth.git
  $ git push -u origin main

STEP 3: GitHub Actions runs automatically
  ✓ Build starts within seconds
  ✓ Builds for ~10 minutes
  ✓ Generates Synth.ipa
  ✓ Upload to Artifacts (always available)
  ✓ Email to blwlego@gmail.com (if SendGrid configured)

═══════════════════════════════════════════════════════════════════════════
TO ENABLE AUTO-EMAIL TO blwlego@gmail.com (OPTIONAL)
═══════════════════════════════════════════════════════════════════════════

1. Create free SendGrid account: https://sendgrid.com
2. Get API key (Dashboard → Settings → API Keys)
3. Add to GitHub repo secrets:
   - Settings → Secrets and variables → Actions
   - New secret: SENDGRID_API_KEY = [your SendGrid key]
4. Next push/build will auto-email the IPA

(Without SendGrid, IPA is still available in Artifacts for download)

  1. Community app store for iOS
  2. Add IPA through interface
  3. Install and refresh
